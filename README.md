## Features

- **Drillcore logging**: Log planes, lines, and fault-slip data with kinematics directly in the app — all you need is your survey data.
- **True orientations on the fly**: Alpha, beta, and gamma converted to true dip, dip direction, trend, and plunge instantly using Minimum Curvature, Balanced Tangential, or Tangential interpolation.
- **Stereonet**: Lower-hemisphere equal-area stereonet with per-type density contouring (Kamb and Gaussian Kamb), fold axis and mean orientation calculations, Terzaghi sampling-bias correction, colour-by column, and interactive filtering.
- **Orientation sets**: Draw freeform polygons on the stereonet to classify measurements into named sets. Full directional statistics per set: Fisher κ, α₉₅, Woodcock K/C, Bingham κ₁/κ₂, and eigenvalue decomposition.
- **3D view**: Interactive OpenGL scene with drillhole traces, structural plane disks, lineation arrows, interval tubes, and point markers.
- **Downhole logs**: Strip-chart log with structural, interval, and point tracks for multiple holes side by side.
- **User-defined tables**: Add interval tables (lithology, RQD, alteration, assay) and point tables alongside your structural data.
- **Import and export**: CSV and Excel import and export for all tables, with 3D coordinates computed automatically on export.
- **Batch export**: Generate stereonet and downhole log images for every drillhole in one operation.
- **Standalone stereonet**: Free with no time limit — import field measurements or any orientation dataset and access the full stereonet toolset without a drillhole project.

## Usage

- Create a new logging project, import your collar and survey data, and start logging — true orientations are computed as you type.
- Use the stereonet, 3D, and downhole views side by side; selecting a measurement in any view highlights it in all others.
- Draw polygons on the stereonet to define orientation sets and compute directional statistics per set and per drillhole.
- Export all data to CSV or Excel at any time, or batch-export figures for all holes in one step.
- Use the standalone stereonet (free, no licence required) for field data or any orientation dataset not tied to drillholes.
